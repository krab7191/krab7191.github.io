About me

Hi, I'm Karsten. I'm a full-stack web developer and FOSS enthusiast. I love Linux and javascript. When I'm not programming for work you can find me learning more about coding, tinkering with a new project on my Raspberry Pi, or learning about a new technology.

What I do

Websites

I bring the latest front end technologies to the table when creating beautiful, mobile-responsive websites. Front end scripting with vanilla JS and jQuery, templating with handlebars, CSS frameworks like Bootstrap, preprocessors like SASS, and even Angular and React.

Web Applications

I build full stack Node and web applications using all the latest tools. I handle the back end with MySQL and sequelize or MongoDB, handle server routing with Express, Apache or NGINX, and render the visualized data with React or Angular. Without a doubt javascript is the name of the game for me.

Open Source

I have exclusively been using Linux systems for almost 10 years now and love and support all open source projects. I have experience with Ubuntu, Mint, Elementary, Kali, Parrot OS, and Raspbian, and have spent years customizing and scripting in these environments. On a constant basis I use Firefox, OpenOffice, GIMP, VLC, Audacity and Thunderbird.

With regards to Android, you can always find me with a rooted device trying out some new ROM. I currently run TWRP and Xenon HD on the Huawei Honor 5X.

Get in touch if you need work done or want to collaborate!

Bamazon

Bamazon is a proof of concept command line mysql storefront application

Bamazon is a backend NodeJS application which implements the packages inquirer, mysql and cli-table. It presents 3 'storefront' portals which interface in different ways with the database to simulate customer, supervisor, and manager responsabilities at this virtual store.

LIRI

LIRI (Language Interpretation and Recognition Interface) is a Node command line media API tool.

LIRI takes user input on the command line and hits the Twitter, OMDB, and Spotify APIs to search for movies, tv shows, songs and tweets.
